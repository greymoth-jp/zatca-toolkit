START IN SECONDS

A few lines. Runs anywhere.

@zatca/sdk · CI-tested · browser or node
// validate an invoice — deterministic, offline, in-process
import { Zatca } from "@zatca/sdk";

const z = new Zatca({ mode: "sandbox" });
const { ubl } = await z.generate(myInvoice);        // normalized -> UBL 2.1
const r = await z.validateXML(ubl, "zatca-ksa");     // ZATCA + EN16931 rules
r.report.valid ? ship(ubl) : fix(r.report.errors);   // each error: rule_id + ar/en + fix

HOW IT WORKS

Generate · validate · clear.

02

Deterministic ZATCA + EN16931 rules. Every failure tells you the fix, in Arabic and English.

03

Sign & clear

secp256k1 XAdES signature; clearance goes through your onboarded credentials or a certified partner.
